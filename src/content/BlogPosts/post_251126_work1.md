這一季在整理與謝野町水田的整套資料時，我開始意識到自己的問題可能問得太小。

原本我以為，施用玄武岩粉末後，一定有某個土壤因子改變，而這個改變能直接帶動 CO₂ 與 CH₄ 的下降。
我主要盯著 pH、Eh、田間含水量和溫度，希望從裡面找出一條明確的線索。

但在反覆檢視後，我發現這四個因子的變化都無法構成能支持「主因—結果」的路徑。
排放量下降的現象是真實的，但土壤性質並沒有給我一個可以收斂的答案。也因此，我開始懷疑：
**是不是我原本就不該期待在田間環境裡找到一條線性的因果關係？**

田間的反應向來是多條路徑同時運作。
所以「沒有找到主因」本身，就是一個結果。

既然常規因子沒有解釋力，我回頭檢查整個處理中最明顯的變化──pH 上升。
施用玄武岩後，pH 的提升非常穩定，這是一個方向一致的訊號。
當 pH 上升時，土壤中的溶解性有機碳（DOC）會進入不同的命運。

DOC 是 CH₄ 與 CO₂ 的前驅物，這一點我之前知道，但我沒有想過要把「DOC 的命運」納入機制推論。

最近的文獻整理中，我看到一個與這季資料非常契合的結果：
**pH 與 Ca²⁺ 的提升，可能把 DOC 從「微生物可利用庫」推向「礦物固定庫」**（Jagadamma et al., 2012）。

Jagadamma 等人提出了兩個重要觀察：
在較高 pH 與較高 Ca²⁺ 的環境中，多羧基的 DOC 分子會更容易與 Ca²⁺ 形成不溶性沉澱（如 Ca–oxalate）。
化學上較穩定的 DOC（例如水楊酸、sinapyl alcohol）在這些條件下幾乎不被微生物分解，因為它們被吸附或沉澱後變得難以被微生物取得。

這兩點加起來的意義是：
當 pH 與 Ca²⁺ 上升時，DOC 會往「沉澱」或「礦物吸附」的方向移動，
從可被利用的池（microbial-available pool）轉移到穩定池。

也就是說，微生物可利用的碳源會降低。
當我把這條路徑重新與田間監測資料疊起來時，一些之前看不懂的結果開始變得合理。

**會不會排放下降，不是環境條件改變，而是反應物變少？**

如果 DOC 因為 pH 上升而更傾向被固定，
那能被微生物使用的 DOC 會下降。
這會引發兩件事：

1. CO₂ 排放下降，因為 DOC 礦化減弱。 
2. CH₄ 排放下降，因為乙酸與 H₂ 的前驅物減少（兩條甲烷生成路徑都受限）。

這樣的機制比「Eh 改變」或「含水量變化」更能解釋田間的結果。

現在看來，CH₄ 不是因為還原條件變弱而下降，而是因為反應物供應不足。
這與過去研究中「水溶性碳濃度與甲烷生成量呈線性關係」的結果**完全一致** (Jugsujinda et al., 1998; Singh et al., 2000)。

這一段時間整理下來，我得到比較可以接受的理解：
> **玄武岩施用後的排放下降是真實且穩定的，但是土壤的 pH、Eh、含水量與溫度不足以解釋下降機制。**

最合理的解釋，是 pH 上升改變了 DOC 的命運，
而不是直接改變土壤環境條件。
這意味著常規指標不夠，後續需要測 DOC、礦物結合有機碳（MAOC）、HCO₃⁻ 等更接近**反應源頭**的指標。
若要理解 ERW 在水田中的作用，跨地區與跨氣候帶的比較很可能是必要的。

這篇整理還不是最終答案，而是記錄。
這些思考會決定下一步的採樣、分析與需要補齊的觀測項目。
